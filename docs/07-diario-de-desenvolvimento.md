# DevLog 001

Hoje percebi que o Tasky deixou de ser apenas um aplicativo de portfólio.

Comecei o projeto pensando apenas em aprender Flutter e construir algo para mostrar no GitHub.

Durante o planejamento, a documentação e as decisões de produto, percebi que estou gostando do processo de desenvolver software de forma organizada.

Pela primeira vez senti satisfação em construir um projeto pensando não apenas no código, mas também na experiência do usuário e na qualidade das decisões tomadas.

Esse é um sentimento que quero manter durante todo o desenvolvimento do Tasky.
