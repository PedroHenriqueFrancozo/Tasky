### RF01 – Autenticação de Usuários

O sistema deve permitir que o usuário realize autenticação utilizando e-mail e senha para acessar sua conta.

#### Regras

- O usuário deve poder criar uma nova conta.
- O usuário deve poder recuperar sua senha.
- O sistema deve validar as credenciais antes de permitir o acesso.

---

### RF02 – Tela Inicial

Após realizar o login, o sistema deve apresentar a tela principal do aplicativo.

A tela inicial deve permitir que o usuário:

- Visualize seu progresso diário.
- Visualize a lista de tarefas.
- Crie uma nova tarefa.
- Navegue entre as principais funcionalidades do aplicativo.

---

### RF03 – Indicadores de Progresso

O sistema deve apresentar informações que permitam ao usuário acompanhar sua evolução.

Os principais indicadores são:

- Percentual de tarefas concluídas no dia.
- Visualização gráfica da produtividade.
- Sequência de dias consecutivos com metas concluídas.
