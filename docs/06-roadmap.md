# Priorização do MVP

Foi utilizada a técnica MoSCoW para definir quais funcionalidades fazem parte da primeira versão do Tasky.

## Must Have

🟢 Must Have (Obrigatório)

Sem isso o produto não existe.

- Login
- Cadastro
- Criar tarefa
- Editar
- Excluir
- Concluir
- Categorias
- Lembretes

---

## Should Have

🔵 Should Have (Importante)

Melhora bastante.

Mas o produto funciona sem.

- Estatísticas
- Perfil
- Alterar senha

---

## Could Have

🟡 Could Have (Seria legal)

- Temas
- Compartilhamento
- Resumo semanal

---

## Won't Have

🔴 Won't Have (Agora não)

- Backend
- Sincronização
- Colaboração
- Permissões
